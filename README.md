# Godz Pack Icons para StreamDock (Rise Mode Stream Deck Vision Sound 01)

## Descrição
O **Godz Pack Icons** é um pacote de ícones moderno e limpo criado por Godz para uso no StreamDock, especialmente compatível com o Rise Mode Stream Deck Vision Sound 01. Ele traz uma grande variedade de ícones para personalizar botões, comandos e ações do seu Stream Deck.

- **Nome:** Godz Pack Icons
- **Versão:** 1.0.0
- **Autor:** Godz

## Estrutura dos Arquivos
- `manifest.json`: Informações do pacote
- `icons.json`: Lista e tags dos ícones
- `icons/`: Pasta com todos os arquivos de ícone PNG
- `icon.png`: Ícone de capa do pacote
- `license.txt`: Licença do pacote

## Instalação
### 1. Copie a pasta do pacote
Coloque a pasta `com.godz.PackIcons.sdIconPack` em:
```
C:\Users\<SEU_USUARIO>\AppData\Roaming\HotSpot\StreamDock\icons\
```

### 2. Adicione o pacote ao StoreCache.json
Abra o arquivo:
```
C:\Users\<SEU_USUARIO>\AppData\Roaming\HotSpot\StreamDock\storecache\StoreCache.json
```
E adicione (ou verifique se já existe) o seguinte bloco na seção `"icons"`:
```json
{
    "device": ["ControllerDeviceS2"],
    "fileName": "com.godz.PackIcons.sdIconPack",
    "localFile": "C:\\Users\\<SEU_USUARIO>\\AppData\\Roaming\\HotSpot\\StreamDock\\icons\\com.godz.PackIcons.sdIconPack",
    "serverFile": "",
    "title": "Pack icons Godz"
}
```
> **Obs:** Se já existir, apenas confirme o caminho.

### 3. Reinicie o StreamDock
Após salvar as alterações, reinicie o StreamDock para que o pacote de ícones apareça na interface.

## Como usar
1. Abra o StreamDock.
2. Vá até a seção de ícones e selecione "Pack icons Godz".
3. Escolha o ícone desejado para personalizar seus botões.

## Observações
- Compatível com dispositivos ControllerDeviceS2 (Rise Mode Stream Deck Vision Sound 01).
- Todos os ícones são em PNG e prontos para uso.
